LocalStorage

to make a variable globally available throughout all the components we use local stoarage .

it store variable inside browser memory. so that every component can access it.

For this we use localStorage class . it has some methods to store and retrieve variables.
localStorage.setItem('key',value) is used to store a variable in browser memory.
localStorage.getItem(variable name) is used to retrieve a variable from browser memory.
localStorage.removeItem(var name) is used to delete a variable in browser memory.
localStorage.clear() is used to delete all variables from browser memory.
